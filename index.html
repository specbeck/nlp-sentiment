About this Project

This project demonstrates cross-lingual sentiment analysis using transfer learning. The model was initially trained on English data and adapted to work with low-resource languages from Uttarakhand, India.

Languages supported:

  • English: Base language with full support
  • Hindi: Adapted using transfer learning
  • Kumaoni: Experimental support for this indigenous language of Uttarakhand
  • Garhwali: Experimental support for this indigenous language of Uttarakhand
